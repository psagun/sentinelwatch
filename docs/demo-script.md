# SentinelWatch — Demo Script (~3 minutes)

---

## Scene 1: Dashboard — The Big Picture (0:00–0:30)

**Screen:** Dashboard page (https://frontend-lilac-eta-55.vercel.app/)

> "Welcome to SentinelWatch — an AI-powered security and compliance monitoring platform. This is the dashboard, your real-time command center. Here you see the overall risk score, severity distribution across critical, high, medium, and low findings, a 14-day trend timeline, and active alerts. Every number here is backed by live Bright Data intelligence and AI analysis."

---

## Scene 2: Adding a Website (0:30–1:15)

**Screen:** Entities page → Click "Add Website" → Enter URL → Open Scan Configuration

> "Let's monitor a website. I'll add **nytimes.com**. SentinelWatch auto-detects the domain name. Now click **Scan Configuration** — here's where the power lies. We can choose which Bright Data products to use: **Web Unlocker** for HTTP scraping that bypasses anti-bot protection, **SERP API** for search intelligence, and **Scraping Browser** for full JavaScript rendering."

> "We can also set **geo-targeting** — I'll pick United States. Toggle **Premium Mode** for hard-to-scrape sites, or **Mobile UA** for mobile-specific results. Then choose compliance frameworks — GDPR, SOC 2, HIPAA, PCI DSS — and specific security checks like SSL validation, security headers, and browser analysis."

> "I'll click **Start Monitoring** and SentinelWatch begins scanning in the background."

---

## Scene 3: Findings & Intelligence (1:15–1:50)

**Screen:** Findings page → scroll through results

> "Once the scan completes, findings appear here. Each finding is classified by severity, enriched with AI-powered risk scoring, and mapped to compliance frameworks. SentinelWatch detects missing security headers, SSL certificate issues, leaked credentials on paste sites, and brand impersonation — all through Bright Data's web intelligence network."

> "The AI Analysis Agent, powered by Claude Opus via AI/ML API, provides remediation recommendations and contextual risk assessment for every finding."

---

## Scene 4: Alerts & Compliance (1:50–2:20)

**Screen:** Alerts page → then Compliance page

> "Critical and high-severity findings generate alerts. You can acknowledge them individually or batch-process them. The alerts page shows the full details with links back to the original findings."

> "The Compliance page maps every finding against regulatory frameworks. Here you can see how findings impact GDPR, SOC 2, HIPAA, and PCI DSS with article-level references."

---

## Scene 5: AI Reports (2:20–2:45)

**Screen:** Reports page → Click "Generate AI Report"

> "SentinelWatch can also generate executive-level AI security reports. These reports include an executive summary, key risk indicators, per-entity risk analysis, compliance impact assessment, a prioritized remediation plan, and strategic recommendations — all powered by Claude Opus."

---

## Closing (2:45–3:00)

**Screen:** Back to Dashboard or presentation title slide

> "SentinelWatch is built with React, TypeScript, FastAPI, and PostgreSQL — deployed on Vercel and Railway at zero monthly cost. Powered by **Bright Data** for web intelligence and **AI/ML API** for AI analysis. The code is open-source on GitHub. Try it yourself at the link below. Thank you!"
